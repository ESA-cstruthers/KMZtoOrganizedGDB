# KMZ Tools

**Owner:** Colin Struthers
**Created:** 2026-05-20
**Last Reviewed:** 2026-05-28
**Status:** QA
**Contact:** Colin S
**Description:** ArcGIS Pro Python toolbox that converts KMZ/KML files into clean, organized File Geodatabases. Two tools: a read-only Inspector for preflight checks, and a one-step KMZ -> Organized GDB converter with a Waterfall UX (container / split-into-FDs / FD source / FC naming) that replaces the older multi-letter mode dropdown. Popup HTML is parsed into typed attribute fields and output is projected to a chosen coordinate system from a curated WGS 1984 / NAD 1983 HARN State Plane / NAD 1983 UTM shortlist.
